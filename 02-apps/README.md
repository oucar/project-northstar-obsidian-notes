# Apps

One subfolder per product lives here. Each app folder follows this structure:

```
02-apps/
└── app-name/
    ├── README.md        # What the app does, target users, current status
    ├── stack.md         # Any deviations from the boilerplate stack
    ├── milestones.md    # App-specific milestone detail (links to 03-milestones)
    └── decisions.md     # App-specific ADRs (links to 04-decisions)
```

## Adding a New App

1. Create a subfolder: `02-apps/<app-name>/`
2. Fill in `README.md` with: description, target users, platform (mobile / web / both), status
3. Note any stack deviations in `stack.md`
4. Add a milestone entry in `03-milestones/README.md`
