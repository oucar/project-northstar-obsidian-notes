# Milestones

Cross-project milestone tracking. Each row is a deliverable with a target date and status.

## Status Key

| Symbol | Meaning |
|---|---|
| `[ ]` | Not started |
| `[~]` | In progress |
| `[x]` | Done |
| `[!]` | Blocked |
